BitTorrent is a peer-to-peer (P2P) file distribution tool written by programmer Bram Cohen and debuted at CodeCon 2002. BitTorrent is designed to distribute large amounts of data without incurring the heavy consumption in costly server and bandwidth resources. The reference implementation is written in Python and is released under the BitTorrent Open Source License (a modified version of the Jabber Open Source License), as of version 4.0. BitTorrent clients implement the BitTorrent protocol which are capable of preparing, requesting, and transmitting any type of computer file over a network using the BitTorrent protocol. The supported files types are text, audio, video, encrypted content, and other types of digital information.
